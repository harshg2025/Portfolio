## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/harshg2025/Portfolio.git
cd Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```

### Production Build

```bash
npm run build
```
